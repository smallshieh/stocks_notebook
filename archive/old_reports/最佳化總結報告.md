# 📈 股票筆記本 — 系統總覽 (更新：2026-02-24)

## 📁 目錄結構

```
股票筆記/
├── 📂 trades/          # 個股交易紀錄（每檔一個 MD）
├── 📂 watchlist/       # 候補股追蹤（觀察中、尚未進場）
├── 📂 journals/        # 每日盤後日誌

├── 📂 strategies/      # 交易策略文件與研究

├── 📄 portfolio_history.csv  # 每日淨值快照（自動追加）
└── 📄 AI_Workflow_SOP.md     # AI 協作使用說明
```

## 🤖 自動化腳本一覽

### ⏰ 每日自動執行（工作日 14:35，Windows 工作排程器）

| 腳本 | 功能 | 輸出 |
|------|------|------|
| `daily_scan.bat` | 🚀 排程總入口，依序執行以下三支腳本 | `journals/logs/YYYY-MM-DD_scan.log` |
| `portfolio_report.py` | 🏥 掃描 29 檔持股的現價、月線、損益，標示跌破月線 / -10% 停損 | `持倉健診_YYYY-MM-DD.md` |
| `portfolio_log.py` | 📊 計算整體總市值與損益，追加一行至歷史記錄；非交易日自動跳過 | `portfolio_history.csv` |
| `watchlist_scan.py` | 🔍 偵測候補股月線/季線位置，觸發條件成立時警示；自動更新 watchlist MD | 終端輸出 + watchlist MD |

### 🛠️ 手動工具（需要時執行）

| 腳本 | 用法 | 功能 |
|------|------|------|
| `stock_analyzer.py` | `--ticker 2330` | 🔎 查單一股票現價、月線、殖利率 |
| `stock_analyzer.py` | `--scan-trades` | ⚠️ 掃描全部 trades 的停損狀態 |
| `vol_check.py` | `--ticker 2002 6488` | 📉 查個股近 3 月成交量、波動幅度（可多檔） |

---

## 🛡️ 腳本健全性

所有腳本均具備：
- ✅ `.TW` / `.TWO` 自動 fallback（上市 / 上櫃皆可抓）
- ✅ 每個交易所 retries 3 次，失敗間隔 5 秒（防短暫網路異常）
- ✅ `warnings` / `logging` 抑制（log 無雜訊）
- ✅ `portfolio_log.py` 假日偵測（以 0050 最新資料日比對今天，非交易日不寫入 CSV）
- ✅ `portfolio_log.py` 資料不完整時不寫入 CSV，避免污染歷史記錄

## 💬 AI 協作

需要 AI 輔助分析時，請參考 **`AI_Workflow_SOP.md`**，裡面有四個情境的標準 Prompt 範本。

> 💡 日常掃描（持倉健診 + watchlist + 淨值記錄）已全自動，**不需要每天叫 AI 跑腳本**，
> 只在需要深度分析、新標的研究、或盤後日誌撰寫時才需要 AI 介入。
