# 零股滾動法區間重算｜使用說明

## 目的
用實際波動資料（20MA、ATR、回檔分位）重算每檔「賣出區 / 買回區 / 暫停線」，避免主觀猜測。

## 腳本位置
- `scripts/quantile_engine.py`（核心歷史分位數決策引擎）
- `scripts/recalc_rolling_ranges.py`（批次重算與策略檔輸出）

## 前置條件
- 必須使用 `.venv`：`S:\股票筆記\.venv\Scripts\python.exe`
- `stocks.csv` 要有代碼對應 ticker（例如 5483 -> 5483.TWO）
- 若 `trades/{code}_*.md` 存在，腳本會自動抓「集保股數 / 買進價格」填入策略檔

## 常用指令

### 1) 更新單一個股（僅顯示，不寫檔）
```powershell
S:\股票筆記\.venv\Scripts\python.exe scripts/recalc_rolling_ranges.py --code 5483
```

### 2) 更新單一個股並回寫策略檔
```powershell
S:\股票筆記\.venv\Scripts\python.exe scripts/recalc_rolling_ranges.py --code 5483 --write
```

### 3) 一次更新多檔並回寫
```powershell
S:\股票筆記\.venv\Scripts\python.exe scripts/recalc_rolling_ranges.py --code 5483,6488 --write
```

### 4) 全部代碼批次更新（較花時間）
```powershell
S:\股票筆記\.venv\Scripts\python.exe scripts/recalc_rolling_ranges.py --all --write
```

### 5) 直接用分析工具輸出「歷史分位數決策診斷」
```powershell
S:\股票筆記\.venv\Scripts\python.exe scripts/stock_analyzer.py --ticker 6488 --quantile
```

## 輸出位置
- 每檔會輸出到：`strategies/零股滾動法_實戰_{code}{name}.md`
- 範例：`strategies/零股滾動法_實戰_5483中美晶.md`

## 價位計算邏輯（摘要）
- 賣出中心：`μ20`（20 日均價）
- 賣出區：`μ20 ± 0.5*ATR14`
- 常規買回區：以賣出中心乘上回檔分位 `dd50 ~ dd70`
- 深回檔區：`dd85`
- 暫停線：`BuyLow - 0.8*ATR14`

## 建議流程
1. 先不帶 `--write` 看輸出區間是否合理
2. 確認後再加 `--write` 回寫策略檔
3. 每完成一輪滾動後重算一次
4. 回寫重要 MD 後，視需要同步 Notion

## 例外與排查
- 顯示 `[SKIP]`: `stocks.csv` 沒有該代碼
- 顯示 `[FAIL]`: 通常是行情抓取失敗或資料不足，稍後重跑
- 若持倉欄位顯示 `（待填）`：代表找不到對應 `trades/{code}_*.md` 或檔內無對應欄位
