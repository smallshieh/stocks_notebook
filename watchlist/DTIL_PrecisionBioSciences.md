## 核心管線

### PBGENE-HBV（慢性 B 型肝炎）— 旗艦

| 項目 | 內容 |
|------|------|
| 試驗名稱 | ELIMINATE-B（Phase 1/2a） |
| 機制 | ARCUS 基因編輯，直接切割 cccDNA 及整合型 HBV DNA |
| 進度 | Cohort 3（最高劑量 0.8mg/kg）全員呈深化抗病毒反應 |
| 關鍵數據 | 劑量依賴性確認；最高劑量組一患者 HBsAg -50% 持續 7 個月；肝臟切片確認基因編輯成功（人類首例） |
| 無劑量限制毒性 | ✅ 已確認（至 0.8mg/kg） |
| 市場規模 | 全球 3 億感染者；功能性治癒市場 >$10B |
| 成功概率（POC）| **~28%**（Bayesian 後驗，已有臨床活性加成） |

### PBGENE-DMD（杜氏肌肉失養症）

| 項目 | 內容 |
|------|------|
| 機制 | ARCUS 永久重構外顯子（exons 45-55） |
| 進度 | IND 已獲 FDA 清除；臨床試驗啟動中 |
| 患者族群 | 涵蓋約 **60%** DMD 患者 |
| 早期終點 | 12 週肌肉切片 dystrophin 表現量 |
| 法規優勢 | FDA **孤兒藥資格** 已取得 |
| 市場規模 | exon-51+ 族群 >$2B；全突變型 TAM 更大 |
| 成功概率（POC）| **~22%** |

**組合機率（至少一項正向）**: ~45%

## 進場計畫（待觸發後執行）

> 微型生技股，建議**總部位不超過整體資產的 5%**，必須使用**限價單（Limit Order）**

| 批次 | 觸發條件 | 參考金額（USD） | 備註 |
|------|---------|----------------|------|
| 第一批（試倉）| 負 EV 確認 + 股價企穩於 $3.8–4.2 區間 | $500–$1,000 | 小額試水，觀察波動 |

**停損參考**：進場後若股價跌破 $3.00（接近帳面現金底部）且有臨床負面消息，停損出場
