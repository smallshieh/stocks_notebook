# 波段操作策略

> 建立：2026-03-27 | 適用倉別：Tactical 倉
> 核心前提：波段邏輯與 B&H 邏輯**不可混用**，進場前先定性。

---

## 第零步：定性——這是波段倉還是 B&H 倉？

在進場前回答以下三題：

```
1. 這家公司 3 年後的護城河比現在更深還是更淺？
2. 現在的進場殖利率 / 本益比，3 年後回頭看會覺得便宜嗎？
3. 就算股價腰斬，我願意繼續持有並加碼嗎？
```

| 三題結果 | 定性 | 對應策略 |
|---------|------|---------|
| 全部「是」| B&H 核心倉 | `長線趨勢股交易策略.md` |
| 有任何「否」或「答不出來」| 波段倉 | **本策略** |

> **東和鋼鐵 2006 教訓（2026-03-27）**：景氣循環股腰斬時會停損而非加碼，第 3 題
> 答不了，應定為波段倉。進場時套用「長期配息倉」邏輯，導致出場條件不清。

---

## 第一步：進場觸發條件（三選一，需明確成立）

進場不是「感覺便宜」，必須對應到下列其中一個觸發類型：

### A. 技術面觸發
```
收盤價 < MA20 × 0.97（月線折價 3%）
且距上次買入 ≥ 20 交易日（避免攤平陷阱）
且成交量無異常放量下殺（確認非崩盤）
```

### B. 估值面觸發
```
現價殖利率 ≥ 歷史均值（近 2 年）
或 P/E 跌至歷史 25% 分位以下
或 OU 模型顯示現價低於均衡價 ≥ 10%
```

### C. 事件面觸發
```
法說會 / 財報後市場過度反應下殺
外資或融資異常賣壓（非基本面惡化）
產業短期利空已充分消化（觀察 3～5 日）
```

**觸發條件未成立 → 不進場，等。**

---

## 第二步：出場條件（進場前就決定好）

出場條件必須與進場條件對稱，不得進場後才設。

### 停損（錯了認賠）

| 類型 | 條件 | 說明 |
|------|------|------|
| 硬停損 | 持倉虧損 ≥ 10% | 無條件執行，不等訊號 |
| 月線停損 | 收盤跌破 MA20 連續 2 日 | 趨勢破壞，波段邏輯失效 |
| 物理警戒線 | OU 均衡價下方 ≥ 15% | 超跌均值仍繼續下行 → 停損 |

### 停利（達標就走）

| 類型 | 條件 | 說明 |
|------|------|------|
| 目標價停利 | 達法人目標價 / OU 阻力壁 | 進場前設定，到點評估量能再決定 |
| 波段回報停利 | 獲利 ≥ 15%（視標的） | 避免貪尾段，果斷出場 |

### 停滯（機會成本停損）

```
持有超過 60 個交易日（約 3 個月），且：
- 股價未達目標價 50%
- Wave Score 沒有改善

→ 認定資金效率低，出場換標的
```

## 第三步：風報比計算（下單前必跑）

```
風報比 = 預期獲利空間 ÷ 最大可接受損失

計算方式：
  預期獲利空間 = 目標價 - 進場價
  最大損失     = 進場價 - 停損價

門檻：風報比 ≥ 2 : 1 才值得進場
```

**範例（東和 2006，實際進場時應計算）：**
```
進場 70.6，目標 77，停損 66.6
預期獲利 = 77 - 70.6 = 6.4 元
最大損失 = 70.6 - 66.6 = 4.0 元
風報比 = 6.4 / 4.0 = 1.6 : 1  → 低於門檻，應減少部位或等更好進場點
```

---

## 配息股額外規範（減持前必算）

賣出配息股前，必須完成以下計算：

```
價差獲利   = (賣出價 - 成本) × 賣出股數
年配息損失 = 配息/股 × 賣出股數
等於幾年配息 = 價差獲利 ÷ 年配息損失
```

| 結果 | 判斷 |
|------|------|
| < 3 年 | 提醒重新考慮，除非有明確理由 |
| 3～7 年 | 合理，結合趨勢判斷 |
| > 7 年 | 明顯划算，支持出場 |

> 東和 2006 實際結果：0.24 年（極低）。波段倉身份下仍合理出場，
> 但若當初正確定性為波段倉，這個計算在設計出場條件時就會發現問題。

## 波段 vs B&H 決策對照

| 情境 | 波段做法 | B&H 做法 |
|------|---------|---------|
| 股價跌 10% | 執行停損 | 評估加碼 |
| 達目標價 | 出場 | 繼續持有 |
| 月線跌破 | 觸發停損評估 | 忽略短線雜訊 |
| 腰斬 | 早已停損出場 | 加碼攤平 |
| 核心問題 | 「現在值得進嗎？」| 「5 年後還更貴嗎？」|

---

## 適用標的特徵

**適合波段的標的：**
- 景氣循環股（鋼鐵、航運、記憶體）
- 技術面驅動、短期催化劑明確
- 估值均值回歸型（殖利率高低驅動）

**不適合波段（應用 B&H）的標的：**
- 護城河深、EPS 長期成長（台積電、台達電）
- 大盤 ETF（0050、00878）——應定期定額
