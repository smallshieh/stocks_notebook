# 四維波段評分加減碼策略（Wave Score 4D）

> 建立日期：2026-03-12
> 工具腳本：`scripts/wave_position.py`
> 適用：所有持倉與候補標的的波段位置判斷、加碼 / 減持決策支援

---

## 核心理念

傳統的「看 K 線決定買賣」方式過於主觀。本策略從四個**量化維度**各自打分，加總後形成客觀的波段位置評分（Wave Score），再對應操作建議，**消除情緒性決策**。

---

## 評分系統

每個維度分數：-2 ～ +2（或 +3）

| 維度 | 工具 | 說明 | 分數範圍 |
|------|------|------|---------|
| **1. 均線結構** | MA 排列 | 現價 > 5MA > 10MA > 20MA > 60MA 各 +1，總分 0~4，基準 -2 | -2 ～ +2 |
| **2. GBM σ 位置** | GBM 漂移模型 | 現價相對 20日 GBM 期望值的 σ 距離 | -2 ～ +2 |
| **3. 分位數區間** | 歷史回檔分位 | 現價在超漲收割 / 合理 / 常規買回 / 深回檔哪一區 | -2 ～ +3 |
| **4. 物理引擎** | 動量 / 雷諾數 / 預警 | 動量方向、流場狀態、反重力 / 能量耗散預警 | -2 ～ +2 |

### 分維度評分細則

**均線結構（MA）：**
| 條件符合數 | 原始分 | 評分 |
|----------|--------|------|
| 4/4（完整多頭排列）| 4 | +2 |
| 3/4 | 3 | +1 |
| 2/4 | 2 | 0 |
| 1/4 | 1 | -1 |
| 0/4（完整空頭排列）| 0 | -2 |

**GBM σ 位置（20日）：**
| 現價區間 | 評分 |
|---------|------|
| < 期望值 - 0.5σ（低估）| +2 |
| 期望值 ± 0.5σ（合理）| 0 |
| 期望值 + 0.5σ ～ +1σ（偏貴）| -1 |
| > 期望值 + 1σ（極端貴）| -2 |

**分位數區間：**
| 現價落點 | 評分 |
|---------|------|
| 深回檔加碼區 | +3 |
| 常規買回區 | +2 |
| 合理持有區 | 0 |
| 超漲收割區 | -2 |
| 低於暫停線 | -3 |

**物理引擎（4 項加總，基準 -2）：**
| 條件 | 說明 |
|------|------|
| 動量正向 ↑ | +1 |
| 雷諾數 < 2000（非極端湍流）| +1 |
| 無反重力預警 | +1 |
| 無能量耗散預警 | +1 |
| 以上每符合一項 +1，總和 -2（基準校正），結果 -2 ～ +2 | |

## 操作建議閾值

| Wave Score | 建議 | 操作方向 |
|-----------|------|---------|
| ≥ +5 | 🟢 強力加碼 | 全力建倉 / 加碼至上限 |
| +3 ～ +4 | 🟢 加碼 | 常規加碼 |
| +1 ～ +2 | 🟡 輕倉加碼 / 觀察 | 試倉或持有 |
| -1 ～ 0 | 🟡 持有不動 | 不動作 |
| -2 ～ -3 | 🔴 部分減持 | 波段倉減碼（底倉不動）|
| ≤ -4 | 🔴 強力減持 | 考慮大幅減倉 |

> ⚠️ **注意**：Wave Score 是輔助工具，**不是交易訊號**。
> 對於長線趨勢股（台積電、鴻海），即使 Score 為負，底倉依然不動；
> Score 指導的是**波段調節倉**的加減碼時機。

---

## 波段位置標籤

| 標籤 | 條件 | 意義 |
|------|------|------|
| 🌊 波段初期 | 現價 ≤ 分位數買點中值 | 回測支撐，新一波可能啟動 |
| 🚀 波段中期 | 現價 > 買點，MA ≥ 3/4 | 趨勢延伸，持有最舒服的階段 |
| 🎯 波段後期 | 現價 ≥ 分位數賣出區 | 接近歷史超漲位，提高警戒 |
| ⚖️  波段中性 | 均線混亂，方向不明 | 等待方向確認 |

## 使用方式

```bash
# 單一標的
python scripts/wave_position.py --code 2330

# 多標的比較
python scripts/wave_position.py --code 2330 --code 2317 --code 3455

# 含新建倉預算試算（預算套用至最後一個 --code）
python scripts/wave_position.py --code 3455 --budget 80000

# 三標的同時分析，含由田預算
python scripts/wave_position.py --code 2330 --code 2317 --code 3455 --budget 80000
```

