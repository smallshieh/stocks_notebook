# Core 部位再平衡建議書（汰弱逐強版）

- **日期**: 2026-03-13（修訂）
- **版本**: v2.0（v1.0 廢棄，改採「汰弱逐強」框架取代「有機稀釋」框架）
- **背景**: Core 部位實際佔比 62.7%，較目標 50% 超配 +12.7%。
  更根本的問題：Core 中存在**不符合入選標準的弱勢持倉**，佔比偏高只是症狀。

---

## 0. Core 入選標準（判斷基準）

一個標的能留在 Core，必須**同時滿足**以下四個條件：

| # | 條件 | 說明 |
|---|------|------|
| 1 | **長期正漂移（GBM μ > 0）** | 不靠交易自然增值；μ 為負代表長期持有是懲罰 |
| 2 | **護城河明確** | 10 年後仍有存在理由，競爭優勢難被取代 |
| 3 | **低管理依賴** | 不需擇時、不需每週盯盤，可放著不動 |
| 4 | **與其他 Core 低相關** | 風險來源不同，真正分散而非表面分散 |

> **負面定義**：有以下任一情形，不得留在 Core：
> - μ < 0（長期自然衰退）
> - 與其他 Core 持倉高度重疊（疊加相同風險）
> - 需要靠時機才能賺錢（本質是 Tactical）
> - 基本面持續惡化、護城河縮窄

## 2. 三階段汰弱逐強計畫

### 現金部署原則（補充）

> 詳細原則見 `capital/capital_config.md` ★ 核心決策原則

**摘要**：
- 所有現金（股利、停損回收、存入）皆同質，部署依**期望報酬最高**原則決定，與來源無關
- 多頭期間子彈優先 Tactical；無明確 Tactical 機會時，Core 好標的仍可加碼
- **Core 比例目標是警戒線**：> 65% 才視為硬限制；50-65% 區間以機會成本判斷，不硬性禁止

> ❌ 錯誤：「這筆是股利，不能買 ETF」
> ✅ 正確：「現在最佳報酬在哪？Core 超配但台積電在低點——加 1 股底倉比讓子彈空置更好」

---

### Phase 0：立即處理（不等回本，依展望判斷）

> **原則**：μ < 0 的持倉，繼續持有是懲罰，不應以買入成本作為出場依據。

| 標的 | 狀態 | 動作 | 說明 |
|------|------|------|------|
| **8069 元太** | ❌ μ = -47.3% | 退場計畫執行中 | 165元/170元或06-30硬死線，詳見 `trades/8069_元太.md` |
| **6115 鎰勝** | ❌ μ = -6.0% | **觀察至 2026-06-30** | 若 μ 仍負 → 在成本附近分批出清；不等回本 |

> ⚠️ Phase 0 回收資金 **100% 進 Cash_Bucket**，待 Cash > 18% 後轉入下列「強」標的。

---

### Phase 1：ETF 汰換（依重疊度優先，績效導向取代成本導向）

**出場觸發條件（擇一即可）**，不再以「回本」作為條件：
- 持有超過 **12 個月**，且年化報酬落後 0050 超過 **5%**
- **μ 轉負**（下次 GBM 審查確認）
- 或：市場回溫時，在合理損益區間（-3% 以內）主動出清

**汰換順序**：

| 順序 | 標的 | 理由 | 回收資金去向 |
|------|------|------|------------|
| 1 | **009816 凱基TOP50** | 與 0050 幾乎完全重疊，無獨立存在理由 | → 直接轉入 **0050** |
| 2 | **00921 兆豐龍頭等權重** | 等權重設計在下跌時尾部風險偏高，與 0050 重疊 | → 直接轉入 **0050** |
| 3 | **00946 群益科技高息成長** | 規模小、費用偏高、與 00919 重疊 | → 轉入 **00919** |
| 4 | **00940 元大台灣價值高息** | 新 ETF 未驗證完整周期，與 00878 重疊 | → 轉入 **00878** |
| 5 | **0056 / 00929 擇一** | 兩者功能重疊，保留 **00919**（高息成長）即可 | → 轉入 **00878 或 0050** |

> **關鍵原則**：每一筆出清，資金直接指定轉入更強的標的（非停留 Cash），這才是「逐強」。

---

### Phase 2：結構優化（逐強，明確目的地）

當 Phase 1 持續執行，Core 應向以下結構收斂：

**目標 Core 結構（50% 組合）**：

| 類型 | 標的 | 目標佔 Core 比例 | 特性 |
|------|------|----------------|------|
| 全市場成長 | **0050** | ~30% | 台灣整體經濟成長，低管理 |
| 高息成長 | **00878 + 00919** | ~25% | 現金流 + 成長，互補 |
| 科技深護城河 | **2330 台積電底倉** | ~15% | μ = +79.6%，AI 最終受益者 |
| 金融防禦 | **2886 兆豐金** | ~10% | 低相關，穩定配息 |
| 消費防禦 | **1215 卜蜂** | ~10% | 食品，抗景氣循環 |
| 其他個股 | 待 GBM 審查 | ~10% | 2493 / 2546 / 6239 確認後配置 |

> **美股方向（第三階段可選）**：建議以 **台灣掛牌的美股 ETF（如 00662 元大標普500）** 為優先，避免複委託稅務複雜度。

## 4. 關鍵原則修訂

| v1.0 舊原則 | v2.0 修訂原則 | 原因 |
|-----------|-------------|------|
| 回本才出 | **展望導向，不錨定成本** | 成本是沉沒成本，未來展望才重要 |
| 股利不再投入（被動） | **出清弱持倉，主動轉入強標的** | 消極等待不是汰弱逐強 |
| 轉向「指數或美股」（模糊） | **每筆出清指定具體去向** | 逐強需要明確目的地 |
| 12-18 個月降至 55% | **24-36 個月降至 50%，含主動汰換** | 時程更現實，目標更精確 |

---

> [!IMPORTANT]
> **執行提醒**：汰弱逐強不等於「一次清倉」。每次市場回溫時，優先執行順序 1-2 的 ETF 汰換；Phase 0 個股（6115、8069）則依各自退場計畫獨立執行，不受市場狀況約束。
