# 交易 Lessons Learned

> **用途**：保存出場後的核心教訓，作為未來決策的輸入。與 `trades/` 完全分離 — `trades/` 只保留當前活躍持倉，已出場的標的在萃取 lesson 後直接 `git rm`，歷史可由 git log 回溯。

## 出場 SOP

1. 標的完全出場後（含停損、停利、換股），先在本檔末尾「出場紀錄」區塊**頂端**新增一筆（依下方格式）
2. `git rm trades/{code}_{名稱}.md`
3. Commit message 格式：`trade: 出場 {code} {名稱}（{損益}），lesson: {一句話}`

> 頂端新增、不追加在底部 — 最新的 lesson 放最上面，翻閱時先看到最近學到的東西。

## 格式範本

    ## {code} {名稱}（持有 YYYY-MM-DD ~ YYYY-MM-DD，實現 {+/-X%}）
    - **入場假設**: 當初為什麼買、期望什麼情境成立
    - **失效點 / 兌現點**: 觸發出場的關鍵信號（停損線、目標價、基本面變化、更好機會）
    - **核心 Lesson**: 一句話的學習重點
    - **下次該怎麼做**: 具體行為修正，避免只停留在「以後要更注意」這種空話

**寫作原則**：
- Lesson 必須可以指導未來行為，不是情緒宣洩
- 若出場決策是**正確的**，也要記錄 — 「做對了什麼」跟「做錯了什麼」同樣重要
- 避免過度自責或過度歸因運氣，專注在**當下可觀察的訊號**與**當下可做的決策**

## 2379 瑞昱（持有 2026-03-06 ~ 2026-04-10，實現 +6.3%，+299 元）

- **入場假設**: 之前 watchlist 已長期觀察，基本面評估 OK；黑天鵝期間 (03-06) 用小量試單（10 股 / 4,760 元）建立追蹤倉位，期待後續加碼或反彈出場。
- **失效點 / 兌現點**: 04-10 反彈至 507~508（接近 1y 區間頂端），μ = -0.9%（區間盤整股）不具長線漂移條件、單檔 4,760 元持續觀察的機會成本高，分兩筆 5+5 連續出清落袋。
- **核心 Lesson**: **基本面 OK 但 μ ≈ 0 的區間股，小金額試單（< 5,000 元）的最大效益就是 +5~10% 落袋**；不要拗成「等真正啟動再加碼」，因為 μ ≈ 0 表示沒有漂移期望，加碼前要先看到 μ 翻正或單一強催化劑。
- **下次該怎麼做**: 試單後若 μ 仍 ≈ 0、月線上下震盪：① 反彈 +5% 直接全清，② 不在試單金額 < 5,000 的部位上設「分批計畫」，③ 若想加碼必須等到 μ 翻正 ≥ +20% 或基本面出現新催化劑（法說/月營收明顯轉強），否則資金留給趨勢股。

## 3455 由田（持有 2026-03-13 ~ 2026-04-09，實現 +33.4%，+66,167 元）

- **入場假設**: AOI 檢測大廠轉型半導體成功（CoWoS/FOPLP 先進封裝 AOI 設備），GTC 2026 催化，毛利率 60% 創十年新高，EPS 10~12 元爆發。μ +46.9% 強趨勢股，法人目標價 165~175 元。
- **失效點 / 兌現點**: 股價從建倉均價 127.68 一路突破 126 體積真空區後急漲至 184，不到一個月衝上法人目標價上緣。04-09 在 182.5 連同底倉一併全數出清 1,000 股，鎖定獲利。觸發邏輯：Wave +2 持續強勢但價格已達 GBM 1 年 47.7% 機率的 180 元目標，且 Cash Override 本金需歸還。
- **核心 Lesson**: **體積真空區 + 高雷諾數的組合是波段暴利的正確場景**。這筆交易驗證了物理模型（Re 2.73 + 126 以上無套牢壓力）的預測力 — 突破後阻力極小，噴發速度超預期。分階段減持（135→150→167.5→182.5）完整吃到主升段，紀律正確。
- **下次該怎麼做**: 當 Re > 2.0 + 體積真空區確認 + μ > +40% 三者同時成立時，**首單可以更大膽**（由田首批才 330 股 / 39,728 元，後來追加到 185k 靠 Cash Override）；但追加必須走聲明式 Cash Override SOP，不要事後補登。分階段減持紀律正確，繼續沿用。

## 1503 士電（持有 2026-02-09 ~ 2026-04-14，實現 -35%，-1,575 元）

- **入場假設**: 對帳單匯入的既有部位（15 股 @299）。μ +48.9% 長線趨勢股，電力基建題材，原本期待靠長期 μ 漂移帶出成本。
- **失效點 / 兌現點**: 買進成本 299 遠高於後續交易區間（180~220），持有量僅 15 股（總成本 4,485 元）導致「等反彈」的機會成本遠高於潛在收益。原分波計畫第一波 240 / 第二波 250 始終未觸發，04-11 補設 04-30 硬死線，04-14 在 194 元（Wave 仍 +0）改走 Layer 1 次日市價清，實現 -35.1%。
- **核心 Lesson**: **少量高成本虧損部位（總成本 < 10k）沒有分批計畫的意義** — 反彈區間 +10% 只回 ~1,000 元，而等待 2 個月的機會成本早超過這個數。分批退場是「大部位保護均價」的工具，不是「小部位拗單」的藉口。
- **下次該怎麼做**: 任何單檔總成本 < 5,000 元的虧損部位，直接市價清光歸零現金桶，不做分波計畫、不設反彈觸發；機會成本優於情緒修復。

## 2376 技嘉（持有 2026-02-09 ~ 2026-04-14，實現 -28.9%，-50,965 元）

- **入場假設**: 對帳單匯入的既有部位（570 股 @332.03 = 189,257 元）。AI 伺服器供應鏈題材、GTC 2026 列台灣 15 家背板股之一。
- **失效點 / 兌現點**: 買進均價過高，股價從 332 一路跌破月線後才啟動減碼：03-16 賣 300 @237.75、03-17 賣 170 @242.5、中間續減，最後 04-14 清剩 40 股 @272 走 Layer 1 收尾。全期加權賣出均價 238.66，-28.9%。
- **核心 Lesson**: **-10% 硬停損是「建倉後」的機械紀律，不是「等反彈出場」的可協商條件**。這筆從 -10% 一路拖到 -28.9%，多出的 -18.9% 虧損是「期待 AI 題材讓價位修復」的情緒稅。題材正確 ≠ 價位正確，題材對但買貴了，結果就是慢性失血。
- **下次該怎麼做**: 建倉後 -10% **機械化市價減半、-15% 剩下減半、-20% 全清**，不再三階段等反彈；既有部位 Wave 轉負 + 跌破月線 ≥ 5 日就執行 Layer 1 次日市價清，不待催化劑。

## 6115 鎰勝（持有 2026-02-09 ~ 2026-04-15，實現 ≈ -1%，約打平）

- **入場假設**: 對帳單匯入的既有配息股底倉（殖利率 6.72%），原本視為長抱型被動現金流來源。
- **失效點 / 兌現點**: 2026-03-13 財報狗三角驗證發現 μ = -6.0%（負漂移）、ROE/EPS 大單依賴且 Q2 幾乎歸零、月營收 YoY 劇烈波動；雖然負債比/流動比健康但不符合 Core 入選標準。改採「留察 → 月線附近分批退場」：03-31 出清 1,000 股、04-13 出清 100 股、04-14 賣 70 股、04-15 賣 50 股全出。原 06-30 硬死線被動能翻正提前觸發（實際在 Wave +1 的反彈區段完成退場）。
- **核心 Lesson**: 高殖利率 ≠ Core 資格。基本面量化審查（μ、EPS 穩定度、護城河）是建倉時的第一關，事後才做審查只能靠留察分批退場打平，完全錯失資金使用效率。
- **下次該怎麼做**: 建倉任何「配息股底倉」前必先跑三角驗證（μ > 0 + 護城河 + 財務健康 + 低管理依賴），四項缺一不建倉；對於既有不合格標的，**不要等硬死線**，Wave 翻正的第一個反彈區就是退場窗口，比等月營收翻轉更有把握。
