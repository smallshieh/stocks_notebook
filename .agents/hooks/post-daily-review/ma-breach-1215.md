## 說明

追蹤 1215 卜蜂收盤價是否連續跌破月線（20MA），達 3 日時觸發警示。

**背景**：2026-04-22 卜蜂收盤 141.0，跌破月線 142.2（-0.8%）。持倉為底倉 800 股 + 波段倉 397 股，月線是波段倉的關鍵防線。

**持倉結構**：
- 底倉 800 股：永不動（除非 μ 轉負或 L5 條件）
- 波段倉 397 股：依 SOP 分層減持（L2/L3/反彈 154 均未觸發）

**判定門檻**：

| 狀況 | 動作 |
|------|------|
| 月線下方 < 3 日 | 觀察，不動作（底倉/波段 SOP 分開判斷）|
| 月線下方 ≥ 3 日 | ⚠️ 評估波段倉是否觸發 L2（Wave ≤ -3 連 5 日）|
| 回到月線上方 | 計數歸零，持有不動 |

### Agent 執行指令（達門檻時強制執行）

當輸出包含「已達 3 日門檻」時，daily-review agent 必須：
1. 在戰術指南 `## P2` 的卜蜂欄位加注「⚠️ 月線下方已達 3 日，檢查 Wave — 若 Wave ≤ -3 → 觸發 L2」
2. 在日誌 `## Hooks` 區塊標記「→ 已更新 P2 卜蜂」
3. **不得自動移入 P1** — 底倉邏輯不同，需人工確認 L2 條件

當輸出包含「月線之上」（計數歸零）時：
1. 移除 P2 的月線警示加注
2. 在 hook 檔名加底線前綴暫停此 hook（`_ma-breach-1215.md`）

**預計存續**：短期 hook，待卜蜂月線情況明朗後暫停。
