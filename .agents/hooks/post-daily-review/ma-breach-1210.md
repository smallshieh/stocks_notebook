## 說明

追蹤 1210 大成收盤價是否連續跌破月線（20MA），達 3 日時觸發警示。

**背景**：2026-04-21 大成收盤 54.10，跌破月線 54.42（-0.8%）。Core 配息底倉，月線是持有的關鍵防線。

**判定門檻**：

| 狀況 | 動作 |
|------|------|
| 月線下方 < 3 日 | 觀察，不動作 |
| 月線下方 ≥ 3 日 | ⚠️ 評估減碼（損益 +6.4%，配息率 5.1% 仍健康）|
| 回到月線上方 | 計數歸零，持有不動 |

### Agent 執行指令（達門檻時強制執行）

當輸出包含「已達 3 日門檻」時，daily-review agent 必須：
1. 將 1210 大成移入戰術指南 `## P1 — 條件觸發即執行`，動作欄填「評估減碼，損益 +6.4%，配息率 5.1% 仍健康，先減 1/3」
2. 在日誌 `## Hooks` 區塊標記「→ 已更新 P1」
3. 不得只記錄警示而不更新戰術指南

當輸出包含「月線之上」（計數歸零）時：
1. 若 1210 在 P1，移回 P2 觀察
2. 在 hook 檔名加底線前綴暫停此 hook（`_ma-breach-1210.md`）

**預計存續**：短期 hook，待大成月線情況明朗後暫停（加底線前綴 `_ma-breach-1210.md`）。
