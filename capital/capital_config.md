## ★ 核心決策原則（最高層級，優先於一切規則）

> **資本配置的唯一標準是期望報酬 × 風險調整，不是維持比例目標。**

1. **機會成本優先**：每一筆現金部署前，先問「所有可投標的中，哪個期望報酬最高？」選最高者，不受桶別限制。

2. **現金來源無差異**：股利、停損回收、薪資存入皆為同質現金。部署決策只看「投入何處報酬最佳」，不看「這筆錢從哪來」。

3. **波段短利 vs 長期複利並重**：Tactical 追求波段報酬，Core 追求時間複利，兩者皆是增加整體投報的手段，依當下市場環境選擇最適工具。

4. **桶別比例是警戒線，不是硬約束**：
   - 比例目標（Core 50% / Tactical 30% / Cash 20%）是**風險警示觸發點**，偏離時提醒檢視，不是禁止動作的規則。
   - 例外：**Core 嚴重超配（> 65%）** 或 **Cash 極度不足（< 10%）** 時，才視為硬性限制。

5. **不作為也有機會成本**：讓現金空置、等待「完美時機」，本身就是放棄了當下最佳期望報酬的選擇。

## 7. Core 部位長期策略（汰弱逐強）

> 詳見 `strategies/Core部位再平衡建議書.md`（v2.0，2026-03-13 修訂）

**入選標準**：μ > 0 + 護城河明確 + 低管理依賴 + 與其他 Core 低相關。四者缺一即汰換。

**現金部署原則**：所有現金（股利、停損回收、存入）皆同質，部署由組合狀態決定，與來源無關。Core 超配（> 50%）期間不加碼 Core；多頭期間子彈優先 Tactical。

1. **Phase 0（即日）**：μ < 0 的持倉不等回本，依展望退場（8069 進行中，6115 觀察至 06-30）

3. **Phase 2**：Core 收斂至「0050 + 00878/00919 + 2330底倉 + 2886 + 1215」結構

**啟動條件**：Cash > 15% 且 Tactical P1 全數執行後啟動 Phase 1。Phase 0 即日起不受此限。
